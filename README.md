# نظام الذكاء الاصطناعي لمؤسسة الحبيب للمستلزمات الطبية

هذا النظام مخصص لتقديم الدعم الفني والمعلومات حول مؤسسة الحبيب للمستلزمات الطبية في اليمن. تم تدريب النظام على بيانات شاملة تتعلق بالخدمات، الوكالات المعتمدة، والمعدات الطبية التي توفرها المؤسسة.

## المميزات
- **قاعدة بيانات مخصصة:** تم إنشاء قاعدة بيانات `alhabib_chatbot.db` تحتوي على بيانات المؤسسة.
- **بوت تلجرام:** بوت متكامل للرد على استفسارات العملاء بشكل آلي.
- **محرك ذكاء اصطناعي:** مدعوم بنموذج Gemini 2.5 Flash لتقديم ردود دقيقة واحترافية.
- **لوحة تحكم المدير:** لمراقبة المحادثات وإدارة بيانات التدريب.

## كيفية التشغيل (الطريقة الأسهل)
لتشغيل الموقع وبوت التلجرام معاً في نفس الوقت، استخدم الملف الموحد:

1. تثبيت التبعات (لأول مرة فقط):
   ```bash
   pip install -r requirements.txt
   ```
2. تشغيل النظام بالكامل:
   ```bash
   python start_all.py
   ```
   *سيقوم هذا الأمر بتهيئة قاعدة البيانات وتشغيل الموقع وبوت التلجرام معاً.*

## التشغيل اليدوي (في نوافذ منفصلة)
إذا كنت تفضل تشغيل كل جزء على حدة:
1. تشغيل الموقع الرئيسي: `python main.py`
2. تشغيل بوت التلجرام: `python telegram_bot.py`

## ملاحظات هامة لإصلاح مشكلة تلجرام:
- تأكد من تشغيل ملف `telegram_bot.py` أو استخدام `start_all.py`.
- البوت يتفاعل تلقائياً مع الرسائل ويستخدم نفس البيانات الموجودة في الموقع.
- تم تحديث ملف `init_and_train.py` و `setup_data.py` لضمان تحميل بيانات مؤسسة الحبيب الطبية بشكل صحيح.
