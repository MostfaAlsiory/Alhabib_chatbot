# تطبيق الذكاء الاصطناعي لبنك الأمل (نسخة مطورة)

هذه هي النسخة المطورة من التطبيق، تم دمجها مع نموذج ذكاء اصطناعي حقيقي وتدريبها ببيانات شاملة عن بنك الأمل للتمويل الأصغر في اليمن.

## المميزات الجديدة:
- **محرك ذكاء اصطناعي حقيقي:** يستخدم نموذج `gpt-4.1-mini` للإجابة على الأسئلة بذكاء بدلاً من مطابقة الكلمات.
- **قاعدة بيانات جاهزة:** تحتوي قاعدة البيانات `instance/ai.db` على 20 زوجاً من الأسئلة والأجوبة المدربة مسبقاً عن بنك الأمل.
- **نظام RAG:** يقوم التطبيق باسترجاع المعلومات من قاعدة البيانات وتزويد نموذج الذكاء الاصطناعي بها ليعطي إجابات دقيقة وموثوقة.

## محتويات الملف:
1. **الكود المصدري:** كافة ملفات Python (Flask).
2. **قاعدة البيانات:** ملف `instance/ai.db` (يحتوي على المستخدمين والبيانات المدربة).
3. **بيانات التدريب:** ملف `alamal_bank_data.txt` (البيانات الخام التي تم استخدامها).

## كيفية التشغيل:
1. تأكد من تثبيت مكتبات Python اللازمة:
   ```bash
   pip install flask flask-sqlalchemy flask-login openai email-validator
   ```
2. قم بضبط مفتاح OpenAI API الخاص بك في متغيرات البيئة:
   ```bash
   export OPENAI_API_KEY='your_api_key_here'
   ```
3. قم بتشغيل التطبيق:
   ```bash
   flask run
   ```
